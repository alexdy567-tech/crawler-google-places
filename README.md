# Crawler Google Places

## Project Repository

[Link to Repository](https://github.com/alexdy567-tech/crawler-google-places)

## Installation
To install the project, follow these steps:
1. Clone the repository:
   ```
   git clone https://github.com/alexdy567-tech/crawler-google-places.git
   ```
2. Navigate to the project directory:
   ```
   cd crawler-google-places
   ```
3. Install the required dependencies:
   ```
   npm install
   ```

## Configuration
Before running the project, ensure that you set up the necessary configurations:
- Create a `.env` file in the root of the project and add your API keys and configurations.

## Usage
To start the application, use the following command:
```bash
npm start
```

## API Documentation
The API endpoints for the project are as follows:
- **GET /places**: Retrieves a list of places based on provided parameters.
- **POST /places**: Adds a new place to the database.
- **GET /places/:id**: Retrieves details of a specific place by ID.

Make sure to check the specific API for the request/response formats and required parameters.

For more detailed usage, refer to the specific sections of the documentation.
